# Analizador de Calidad de Datos para IA
 Este proyecto es parte de la asignatura ITY1101. Consiste en una herramienta para validar la integridad de datasets antes de ser procesados por modelos de IA.

## Entorno Técnico
**Editor:** VS Code / GitHub Codespaces.

**Contenedor:** Docker.

**Base de Datos:** Supabase (PostgreSQL).

**CI/CD:** GitHub Actions.
